This repository contains the data and MATLAB code required to reproduce the behavioral and computational analyses reported in:

Wu, X., Fu, H., Aydogan, G., Feng, C., Qin, S., Zeng, Y., & Liu, C. (2025). The Self-Interest of Adolescents Overrules Cooperation in Social Dilemmas. eLife, 14.

All analyses were developed and tested using MATLAB R2023a (MathWorks, Natick, MA). The repository includes anonymized datasets and fully reproducible scripts.
